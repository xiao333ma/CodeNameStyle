# CodeNameStyle

大驼峰，小驼峰，下划线之间的转换

```sh
./NameStyle -f xh-dx -s CodeNameStyle 

CODE_NAME_STYLE
```

```shell
./NameStyle -f xtf -s CODE_NAME_STYLE   

codeNameStyle
```

Usage of ./NameStyle:
  -f string
        
        指定方式
